## Turn AI Coding Into an Engineering Process

Gem Team wraps your AI with a structured workflow: plan, build, review, debug, learn: so every task follows good engineering practice by default.

**Why it matters:**

- **Quality by Default**: TDD, code reviews, and security audits happen automatically
- **Smart & Efficient**: fewer tokens, lower costs, no context bloat
- **Works With Your Tools**: Copilot, Claude, Cursor, Codex, Gemini, Windsurf
- **Learns & Improves**: remembers what works, extracts reusable skills

Under the hood, a set of **agent definitions** enforces good software engineering practices. An Orchestrator routes work, manages context, and enforces verification gates: so quality is never skipped.

> **See it in action**: [Workflow →](/guide/workflow) covers the full pipeline. [Features →](/guide/features) covers all capabilities. [Comparison →](/guide/comparison) shows how it differs from other tools.

## How It Works (Simple View)

```text
Your Request
    ↓
Understand & Plan     ← Clarify intent, pick the right approach
    ↓
Execute & Verify      ← Build with TDD, review automatically, test
    ↓
Learn & Improve       ← Save patterns, skills, decisions for next time
    ↓
Done                  ← Status report + reusable knowledge
```
