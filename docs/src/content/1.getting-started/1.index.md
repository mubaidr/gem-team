## What is Gem Team?

Gem Team is a **spec-driven multi-agent orchestration system** for software development. It wraps your AI with a disciplined engineering delivery pipeline: plan, build, review, debug, and learn — all within a structured, resumable workflow.

It installs **16 specialist agents** that turn AI coding into an engineering process. Each agent has a focused role — planning, implementation, review, debugging, design, DevOps, documentation, and more — coordinated by an Orchestrator that routes work, manages context, and enforces verification gates.

### Key Differentiators

- **Better delivery flow**: Spec-driven execution, wave-based parallelism, verification gates, resumable plans.
- **Better code quality**: 16 specialist agents, TDD by default, diagnose-then-fix, security and accessibility audits.
- **Better context management**: Progressive context envelope, three-tier memory, skill extraction — context bloat avoidance built in.
- **Better cost control**: Model routing, output hygiene, context pruning, discovery depth scaling — fewer tokens, same results.
- **Better tool calling**: Targeted context snapshots per agent, output hygiene rules — precision without prompt waste.

Gem Team operates on a **complexity-adaptive pipeline**:
