## What Is Gem Team?

Gem Team is a **spec-driven multi-agent orchestration system** for software development. It wraps your AI with a disciplined engineering pipeline: plan, build, review, debug, learn — all within a structured, resumable workflow.

It installs **16 specialist agents** (planner, implementer, reviewer, debugger, designer, DevOps, and more) coordinated by an Orchestrator that routes work, manages context, and enforces verification gates. Each agent has a focused role and never executes outside its domain.

> **See it in action**: [Workflow →](/guide/workflow) covers the full pipeline. [Features →](/guide/features) covers all capabilities. [Comparison →](/guide/comparison) shows how it differs from other tools.

## Architecture Flow

```text
User Input
    ↓
Phase 0: Init & Clarify     ← Read context, detect intent, classify complexity
    ↓
Phase 1: Route              ← Continue plan, revise plan, or start new task
    ↓
Phase 2: Plan               ← TRIVIAL/LOW checklist or MEDIUM/HIGH durable plan
    ↓
Phase 3: Execute            ← Wave-based delegation to specialist agents
    ↓
[Learn & Persist]           ← Save decisions, patterns, gotchas, skills
    ↓
[Loop / Replan]             ← Next wave, replan if scope changes, or escalate
    ↓
Phase 4: Output             ← Status report with progress & next steps
```
