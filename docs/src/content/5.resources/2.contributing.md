## Contributing

Contributions are welcome. Please read [CONTRIBUTING.md](https://github.com/mubaidr/gem-team/blob/main/CONTRIBUTING.md) before opening a pull request.

### Recommended Flow

1. **Open or pick an issue** from the [issue tracker](https://github.com/mubaidr/gem-team/issues).
2. **Create a focused branch** for your changes.
3. **Keep changes small and reviewable** — one feature or fix per PR.
4. **Add or update tests/docs** where relevant.
5. **Run validation** before opening the PR.
