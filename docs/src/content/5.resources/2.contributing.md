Contributions are welcome. Please read [CONTRIBUTING.md](https://github.com/mubaidr/gem-team/blob/main/CONTRIBUTING.md) before opening a PR.

## Recommended Flow

1. **Pick or open an issue** from the [issue tracker](https://github.com/mubaidr/gem-team/issues).
2. **Create a focused branch** — one feature or fix per PR.
3. **Add or update tests/docs** where relevant.
4. **Run validation** before opening the PR.
