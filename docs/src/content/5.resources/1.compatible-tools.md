## APM Targets

APM writes different files depending on the selected target and the primitives included in the package.

| Target     | Tool / Harness                       | Typical Output                                                                         |
| :--------- | :----------------------------------- | :------------------------------------------------------------------------------------- |
| `copilot`  | VS Code Copilot / GitHub Copilot CLI | `.github/agents/`, `.github/instructions/`, `.github/prompts/`, and VS Code MCP config |
| `claude`   | Claude Code                          | `.claude/agents/`, `.claude/rules/`, commands, skills, hooks, and MCP config           |
| `cursor`   | Cursor                               | `.cursor/agents/`, `.cursor/rules/`, skills, commands, hooks, and MCP config           |
| `opencode` | OpenCode                             | `.opencode/agents/`, commands, skills, MCP, and compiled instructions                  |
| `codex`    | Codex CLI                            | `.codex/agents/`, `AGENTS.md`, and Codex config                                        |
| `gemini`   | Gemini CLI                           | `GEMINI.md`, skills/instructions, and Gemini config                                    |
| `windsurf` | Windsurf / Cascade                   | `.windsurf/rules/`, skills, commands, hooks, and MCP config                            |

Some harnesses do not support every primitive. APM compiles or skips unsupported primitives according to the target.

## Marketplace Installation

APM is the recommended installation path. Direct marketplace installs are optional.

### GitHub Copilot CLI

```bash
copilot plugin marketplace add mubaidr/gem-team
copilot plugin marketplace browse gem-team
copilot plugin install gem-team@gem-team
```

### Claude Code

```bash
/plugin marketplace add mubaidr/gem-team
/plugin
/plugin install gem-team@gem-team
/reload-plugins
```

### awesome-copilot

For VS Code users, [awesome-copilot](https://github.com/mubaidr/awesome-copilot) is an alternative marketplace that provides manual plugin installation via VS Code extensions. Browse available packages or install directly through the VS Code marketplace.
