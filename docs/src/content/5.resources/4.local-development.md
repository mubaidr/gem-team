## From Source

Clone the repository and install it into a test project:

```bash
git clone https://github.com/mubaidr/gem-team.git
cd gem-team
apm install . --target claude,cursor --dry-run
```

Then run a real install from the local path:

```bash
apm install /absolute/path/to/gem-team --target claude,cursor
```

## Package Validation

For package authoring and release validation:

```bash
apm audit
apm compile --target copilot,claude,cursor --validate
apm pack
```

## Docs Site

The documentation site (this site!) lives at `docs/src/` in the repository. It's a Nuxt 4 application with Nuxt UI and Nuxt Content.

```bash
cd docs/src
npm install
npm run dev
```

> Refer to the [Nuxt Content documentation](https://content.nuxt.com) for MDC syntax reference and content authoring.
